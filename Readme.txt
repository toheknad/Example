1. Controllers - контролеры, каждый в своём каталоге. 
	В корне Default.php контролер, куда будут сваливатся
	все запросы которые не понятные, то есть запросы к несуществующим контролерам.

2. Templates - шаблоны. Для каждого контролера в своей одноименной контролеру папке. 
	В корне главные, обшие шаблоны по умолчанию.	

3. Web - ресурсs для страниц. Java скрипты, стили и т.д. Всё по папочкам

4. Models - модели доступа к БД, все в куче в одной папке.

5. Lib - общие библиотеки, которые будем таскать по всем сайтам. То есть наш движок.

6. Config - конфигруационные файлы. Для каждого режима свой.
